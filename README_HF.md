# Gousto Meal Planner

A full-stack meal planning application built on a recipe database scraped from Gousto's cookbook.

## Features

- **Recipe Browser** - Browse and search 1000+ recipes with filters
- **Meal Planner** - Drag-and-drop weekly meal planning
- **Nutrition Dashboard** - Track macros and nutritional goals
- **Shopping List** - Auto-generated shopping lists from meal plans
- **Favorites** - Save and organize your favorite recipes

## Tech Stack

- **Backend**: Python, FastAPI, SQLAlchemy, SQLite
- **Frontend**: React 18, TypeScript, Vite, TanStack Query, Tailwind CSS
